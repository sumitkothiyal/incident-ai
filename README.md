# ⚡ OpsGPT — AI Incident Command Center

OpsGPT is a multi-agent AI system that simulates real-world incident response using Claude.

## 🔥 Features
- 🧑‍💼 Support Agent (customer response)
- 📊 Analytics Agent (root cause analysis)
- 👨‍💻 Dev Agent (fix recommendations)
- ⚡ ChatOps-style UI
- 🚨 Severity detection (P1 / P2 / P3)

## 🧠 Architecture
User → UI (Netlify) → API (Render) → Claude AI

## 🚀 Live Demo
- UI: https://opsgpt.netlify.app/
- API: https://incident-ai.onrender.com

## 💡 Use Case
Designed for enterprise incident management, DevOps, and AI-driven operations.
